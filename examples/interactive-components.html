Tabs

Overview

Tabs switch between panels using data-ct-target pointing to panel IDs.

Code Example

<div class="tabs">
  <button class="tab active" data-ct-target="#panel-1">Tab 1</button>
  <button class="tab" data-ct-target="#panel-2">Tab 2</button>
</div>

What is Corrupted Theme?

A visual aesthetic NPM package simulating neural corruption, data degradation, and system instability.

How do I install it?

npm install @whykusanagi/corrupted-theme

Does it require JavaScript?

CSS works standalone. JavaScript adds interactivity (accordion toggle, modal, dropdown, etc.).
